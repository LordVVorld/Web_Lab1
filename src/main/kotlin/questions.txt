9) Чем val отличается от var ?

11) Какую конструкцию можно использовать вместо конструкции if ?
12) Как в Kotlin происходит делегирование ?

1) Конструктор - это метод класса, вызываемый при создании экземпляра класса и инициализирующий его свойства.
2) Метод - это функция класса, выполняющая некоторые действия с объектом класса или его свойствами.
3) Дата-класс (data class) - это класс, предназначенный для хранения данных.
4) Интерфейс - это абстрактный класс, который содержит только абстрактные методы и константы.
5) Kotlin поддерживает множественное наследование интерфейсов, то есть класс может реализовывать несколько интерфейсов.
6) Один интерфейс может наследовать несколько других интерфейсов.
7) Оператор try-catch используется для обработки исключений, возникающих во время выполнения программы.
8) Названия объектов принято писать с использованием camelCase. Названия классов принято писать с заглавной буквы каждого слова.
9) Val - ключевое слово для объявления неизменяемых переменных. Var - ключевое слово для объявления изменяемых переменных.
10) Множество (Set) содержит уникальные элементы и порядок элементов не имеет значения.
11) Вместо конструкции if можно использовать оператор when.
12) Делегирование в Kotlin происходит за счет использования интерфейсов делегирования.
    При этом объектом, на который делегируется работа, является другой объект, который реализует соответствующий интерфейс.
